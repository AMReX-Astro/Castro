Derive_nd.F90¶

derive_module¶

Description

All subroutines in this file must be threadsafe because they are called inside OpenMP parallel regions.

Quick access

Routines:derabar(), derangmomx(), derangmomy(), derangmomz(), dercond(), derdiffcoeff(), derdiffterm(), derdivu(), dereint1(), dereint2(), derentropy(), derenuctimescale(), dergamma1(), derkineng(), derlogden(), dermachnumber(), dermaggrav(), dermagmom(), dermagvel(), dermagvort(), derpres(), derradialvel(), dersoundspeed(), derspec(), derstate(), deruminusc(), deruplusc(), dervel()

Needed modules

  • castro_error_module
  • amrex_fort_module (rt() => rt())
