function interpolate_module/interpolate(r, npts_model, model_r, model_var[, iloc])¶

given the array of model coordinates (model_r), and variable (model_var), find the value of model_var at point r (var_r) using linear interpolation. Eventually, we can do something fancier here.

Parameters:
  • r [real,in]
  • model_r (npts_model) [real,in]
  • model_var (npts_model) [real,in]
  • iloc [integer,in,]
Options:

npts_model [integer,in,optional/default=len(model_r)]
Return:

interp [real]
Use :

amrex_fort_module (rt() => rt())
Call to:

locate()
subroutine interpolate_module/tri_interpolate(x, y, z, npts_x, npts_y, npts_z, model_x, model_y, model_z, model_var, interp_var, derivs, error)¶

tri-linear interpolation; useful for EOS tables this is stricly interpolation, so if the point (x,y,z) is outside the bounds of model_x,model_y,model_z, then we abort

Parameters:
  • x [real,in]
  • y [real,in]
  • z [real,in]
  • model_x (npts_x) [real,in]
  • model_y (npts_y) [real,in]
  • model_z (npts_z) [real,in]
  • model_var (npts_x,npts_y,npts_z) [real,in]
  • interp_var [real,out]
  • derivs (3) [real,out]
  • error [logical,out]
Options:
  • npts_x [integer,in,optional/default=len(model_x)]
  • npts_y [integer,in,optional/default=len(model_y)]
  • npts_z [integer,in,optional/default=len(model_z)]
Use :

castro_error_module, amrex_constants_module (one()), amrex_fort_module (rt() => rt())
