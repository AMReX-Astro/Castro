thermo_nd.F90¶

thermo_sources¶

Description

Functions for implementing the source terms to the internal energy equation, for those integration methods where we treat it as a source and do not explicitly discretize it in the conservative update.

Quick access

Routines:ca_thermo_src()

Subroutines and functions

subroutine thermo_sources/ca_thermo_src(lo, hi, old_state, os_lo, os_hi, new_state, ns_lo, ns_hi, src, src_lo, src_hi, problo, dx, time, dt)¶
Compute thermodynamic sources for the internal energy equation.
At the moment, this is only the -p div{U} term in the internal energy equation, and only for method-of-lines integration, including the new SDC method (the - is because it is on the RHS of the equation)
Parameters:
  • lo (3) [integer,in]
  • hi (3) [integer,in] :: bounds of the box to operate on
  • old_state (os_hi(1)-(os_lo(1))+1,os_hi(2)-(os_lo(2))+1,os_hi(3)-(os_lo(3))+1,nvar) [real,in]
  • os_lo (3) [integer,in]
  • os_hi (3) [integer,in] :: bounds of the old_state array
  • new_state (ns_hi(1)-(ns_lo(1))+1,ns_hi(2)-(ns_lo(2))+1,ns_hi(3)-(ns_lo(3))+1,nvar) [real,in]
  • ns_lo (3) [integer,in]
  • ns_hi (3) [integer,in] :: bounds of the new_state array
  • src (src_hi(1)-(src_lo(1))+1,src_hi(2)-(src_lo(2))+1,src_hi(3)-(src_lo(3))+1,nvar) [real,inout]
  • src_lo (3) [integer,in]
  • src_hi (3) [integer,in] :: bounds of the src array
  • problo (3) [real,in] :: physical coordinates of the lower left corner of the domain
  • dx (3) [real,in] :: grid spacing
  • time [real,in,value] :: current simulation time
  • dt [real,in,value] :: current timestep
