bc_ext_fill_nd.F90¶

bc_ext_fill_module¶

Description

this module contains different routines for filling the hydrodynamics boundary conditions

Quick access

Routines:ext_denfill(), ext_fill(), ext_gravxfill(), ext_gravyfill(), ext_gravzfill()

Needed modules

  • amrex_constants_module (zero(), half())
  • castro_error_module (castro_error())
  • amrex_fort_module (rt() => rt())
  • meth_params_module (nvar(), urho(), umx(), umy(), umz(), ueden(), ueint(), ufs(), utemp(), const_grav(), hse_zero_vels(), hse_interp_temp(), hse_reflect_vels(), xl_ext(), xr_ext(), yl_ext(), yr_ext(), zl_ext(), zr_ext(), ext_hse(), ext_interp())
  • prob_params_module (dim()): This module stores the runtime parameters that define the problem domain. These parameter are initialized in set_problem_params().
