subroutine sdc_util/sdc_newton_subdivide(dt_m, u_old, u_new, c, sdc_iteration, err_out, ierr)¶

This is the driver for solving the nonlinear update for the reacting/advecting system using Newton’s method. It attempts to do the solution for the full dt_m requested, but if it fails, will subdivide the domain until it converges or reaches our limit on the number of subintervals.

Parameters:
  • dt_m [real,in]
  • u_old (nvar) [real,in]
  • u_new (nvar) [real,inout]
  • c (nvar) [real,in]

subroutine sdc_util/sdc_vode_solve(dt_m, u_old, u_new, c, sdc_iteration)¶
