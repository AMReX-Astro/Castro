trace_ppm.F90¶

trace_ppm_module¶

Description

These routines do the characteristic tracing under the parabolic profiles in each zone to the edge / half-time.

Quick access

Routines:trace_ppm(), trace_ppm_gammae(), trace_ppm_rhoe(), trace_ppm_species(), trace_ppm_temp()

Needed modules

  • prob_params_module (dg()): This module stores the runtime parameters that define the problem domain. These parameter are initialized in set_problem_params().
  • castro_error_module
  • amrex_fort_module (rt() => rt())
  • amrex_constants_module (zero(), half(), one())
