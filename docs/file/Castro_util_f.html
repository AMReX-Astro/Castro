function castro_util_module/area(i, j, k, dir)¶

Given 3D indices (i,j,k) and a direction dir, return the area of the face perpendicular to direction d. We assume the coordinates perpendicular to the dir axies are edge-centered. Note that Castro has no support for angular coordinates, so this function only provides Cartesian in 1D/2D/3D, Cylindrical (R-Z) in 2D, and Spherical in 1D.

Parameters:
  • i [integer,in]
  • j [integer,in]
  • k [integer,in]
  • dir [integer,in]
Return:

arear [real]
